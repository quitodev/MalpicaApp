# MalpicaApp

MalpicaApp es una aplicación vinculada a un sistema de control de stock que desarrollé hace algún tiempo (MalpicaSoft), el cual permite tener un control de las compras y ventas
que realiza un comercio.

Con la aplicación, podés realizar prácticamente las mismas operaciones que si estuvieras utilizando el software, sumándole la posibilidad de tener toda la información, no sólo en la PC, sino también en un smartphone.

El proyecto está finalizado pero no está publicado en ninguna tienda, debido a que prioricé trabajar en otros proyectos que tenía en paralelo.

## Tecnologías implementadas
- JAVA
- MySQL
